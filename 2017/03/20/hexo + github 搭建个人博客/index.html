hexo + github 搭建个人博客

环境配置

  • 安装Node（必须）

用来生成静态页面

到Node.js官网下载对应平台的最新版本即可

  • 安装Git（必须）

将hexo提交到github上

搭建过程

申请github账号，新建repository，例如github账号为xiaoleo911，repository名就为xiaoleo911.github.io
新建博客目录如tinkfu，然后进入目录安装hexo 

sudo npm install -g hexo

初始化

hexo init

配置
在新建的blog目录下打开_config.yml文件，在最下面修改成这个样子

deploy:
  type: git
  repo: git@github.com:xiaoleo911/xiaoleo911.github.io.git
  branch: master

执行以下代码

npm install hexo-deployer-git --save
hexo generate
hexo deploy

这样页面就上传到github上了，可以通过访问xiaoleo.github.io来访问博客了，这里的xiaoleo911是我的github账号，换成自己的即可，据说上传页面之后有5分钟的延迟，才可以开发。

还可以输入

hexo server -p 4000

在本地浏览器输入localhost：4000来访问blog，如果端口被占用可以将4000换成其他的端口，我的端口就被一个看pdf的软件占用了。。。
